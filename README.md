# Eventsignup Backend
Used technologies:

- Java 17
- Springboot 2.6.x
- Apache IO Commons
- Gradle
- MongoDB 5.0.5
- JUnit 5 & Mockito
- Project lombok

## Development

- Import existing sources as a new project in your favorite IDE (e.g. IntelliJ IDEA)
- Import Gradle project
- Expects Mongodb at localhost:27017
- Server will be at localhost:8080

To run:

    $ ./gradlew bootRun
